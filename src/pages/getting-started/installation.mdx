# Installation

Follow the instructions below to install DJS-Commands and the other required Dependencies.

<Steps>
### Install Required Dependencies

Ignore the ones you already have installed.

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab>
    ```bash copy
    npm install discord.js dotenv
    ```
  </Tab>
  <Tab>
    ```bash copy 
    yarn install discord.js dotenv
    ```
  </Tab>
  <Tab>
    ```bash copy 
    pnpm install discord.js dotenv
    ```
  </Tab>
</Tabs>
### Install DJS-Commands

<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab>
    ```bash copy
    npm install @d3oxy/djs-commands@latest
    ```
  </Tab>
  <Tab>
    ```bash copy 
    yarn install @d3oxy/djs-commands@latest
    ```
  </Tab>
  <Tab>
    ```bash copy 
    pnpm install @d3oxy/djs-commands@latest
    ```
  </Tab>
</Tabs>
### Install Optional Dependencies (Recommended)
<Tabs items={["npm", "yarn", "pnpm"]}>
  <Tab>

  </Tab>
  <Tab>

  <Tab>

</Steps>
